---
name: status
description: Check InTandem workspace status — connection health, online peers, and workspace info. Use when the user asks if they're connected or who's online.
---

Check InTandem status:

1. Call `intandem_peers` to see who's online and connection health.
2. Call `intandem_board` to see task progress.
3. Report: workspace name, your username, connected peers, task summary, connection health.
4. If not connected, suggest `intandem_rejoin` or `intandem_create`.
